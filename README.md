# Burrow

Burrow is an abandoned communication platform inspired by applications such as Discord, with a focus on privacy, minimal account requirements, and self-hostable infrastructure.

The project includes groundwork for web and mobile clients, real-time messaging, voice communication, video streaming, and account synchronization. Burrow was designed around the idea that users should not be required to provide personal information such as email addresses or phone numbers simply to communicate online.

A running version of the platform can be accessed here:

## Features

* Real-time messaging
* Voice channels and audio streaming
* Video and camera support
* Mobile and web clients
* TURN/STUN support through CoTURN
* Minimal user data collection
* Username-based accounts
* Hardware-backed authentication concepts (security keys, passkeys, Windows Hello, YubiKey, etc.)
* Account synchronization across devices

## Project Status

This project is no longer under active development.

Development stopped after portions of the original source and supporting materials were lost. While much of the groundwork remains, I no longer have the interest, time, or project direction needed to continue development.

The repository is being preserved for historical and educational purposes.
